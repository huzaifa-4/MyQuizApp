🧠 QuizMaster – Modern Android Quiz Application

QuizMaster is a feature-rich Android quiz application developed using Kotlin and XML, demonstrating modern Android development practices including API integration, MVC architecture, responsive UI design, and activity navigation.

The application dynamically fetches quiz questions from the Open Trivia Database (OpenTDB) API and includes a fallback system with static questions to ensure reliability.

🚀 Key Features
Multiple-choice quiz system (5+ questions)
Real-time score calculation
Navigation (Next / Previous)
Results screen with performance feedback
Restart quiz functionality
Image support for questions
Dynamic API-based question loading
Error handling with fallback system
Professional UI design
Clean architecture (MVC)
